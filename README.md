В .env вбейте ваш токен, после чего запустите мейн. Загрузите ваш дневник сделок, поторгуйте пару тикеров или просто понаблюдайте - создастся ДБ. Далее из корня запустите train_torch.once, что позволит на исторических данных поучить нейросеть. Вуаля - вы восхитительны. Не является индивидуальной инвестиционной рекомендацией. При форках - пожалуйста, кредитуйте.
